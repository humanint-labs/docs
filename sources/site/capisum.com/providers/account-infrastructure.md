# Account infrastructure integrations

Provision accounts, link bank data, and initiate pay-by-bank flows through standardized Capisum adapters. Connect Mono, Okra, Anchor, Bridge, and other account infrastructure providers from one integration layer.

Supported providers in this category include Bridge, Unit, Treasury Prime, Mono, Okra, Stitch, Anchor, and Maplerad. Capisum is an integration layer — it standardizes API requests and responses without replacing your provider accounts or holding funds.
