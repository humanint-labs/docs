# Payment gateway integrations

Connect Nigeria and Africa's leading payment gateways through one Capisum API. Route pay-in, payout, and virtual account flows across Paystack, Flutterwave, Monnify, and more without rebuilding each integration.

Supported providers in this category include Paystack, Flutterwave, Monnify, Interswitch, Remita, Korapay, SeerBit, Squad, and Fincra. Capisum is an integration layer — it standardizes API requests and responses without replacing your provider accounts or holding funds.
