The Capisum **customer API** (`/v1/*`) is what your backend calls to create payouts and query transactions. Authenticate every request with an API key:

```
Authorization: Bearer cps_your_key_here
```

Base URL: `https://api.capisum.com`

## Endpoints

| Method | Path | Description |
| --- | --- | --- |
| `POST` | `/v1/transactions/payout` | Create a payout |
| `GET` | `/v1/transactions` | List transactions |
| `GET` | `/v1/transactions/ref/{reference}` | Get by reference |
| `GET` | `/v1/transactions/{id}` | Get by ID |
| `POST` | `/v1/transactions/{id}/verify` | Poll provider for status |
| `GET` | `/v1/providers` | List available provider slugs |
| `GET` | `/v1/providers/{slug}/status` | Provider health |
