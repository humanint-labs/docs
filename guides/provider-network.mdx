## Connect a provider

1. Create an account with the underlying provider and obtain API credentials.
2. Open the [Capisum dashboard](https://app.capisum.com) and connect the provider for staging or production.
3. Configure routing rules and call [`POST /v1/transactions/payout`](/api-reference) (or future pay-in endpoints as they ship).

New adapters are announced in the docs and dashboard as they become connectable. This list describes the integration landscape Capisum targets—it is not a statement that every provider is available in your dashboard today.
