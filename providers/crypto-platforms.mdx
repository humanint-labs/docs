Crypto platforms each expose their own proprietary API schemas for buying, selling, transferring assets, and settling to fiat. Capisum normalizes all of that into a single request and response model, so your application does not need to handle provider-specific crypto API differences directly. Whether you're building a crypto-enabled checkout, a settlement layer, or a wallet product, you interact with the same Capisum objects regardless of which exchange or platform is executing the operation.

<Note>
  Capisum standardizes the request and response for crypto operations — your app does not need to handle provider-specific crypto API differences. You still need your own account and credentials with each crypto platform. Connect them under **Settings → Providers** in the Capisum dashboard.
</Note>

## Unified request shape

Crypto operations use the same `capisum.payments.create()` method as other categories. The `provider` field determines which platform executes the operation, and the `type` field specifies the operation you want to perform.

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    // Crypto settlement via Quidax
    const transaction = await capisum.payments.create({
      provider: "quidax",
      type: "pay_in",
      amount: 50000,
      currency: "NGN",
      customer: { email: "customer@example.com" },
      reference: "crypto-settlement-001",
    });
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    curl -X POST https://api.capisum.com/v1/transactions \
      -H "Authorization: Bearer <YOUR_API_KEY>" \
      -H "Content-Type: application/json" \
      -d '{
        "provider": "quidax",
        "type": "pay_in",
        "amount": 50000,
        "currency": "NGN",
        "customer": { "email": "customer@example.com" },
        "reference": "crypto-settlement-001"
      }'
    ```
  </Tab>
</Tabs>

## Live providers

These providers are production-ready. Their adapters are fully implemented and available for live traffic.

---

### Quidax **Live**

Connect crypto buy, sell, and settlement workflows. Quidax is a regulated Nigerian crypto exchange that supports NGN on-ramp and off-ramp alongside a range of digital assets.

**Supported operations:** Crypto, Settlement

---

### Binance **Live**

Access exchange-linked crypto pay and transfer flows. Binance's Pay and wallet infrastructure provides access to a large global liquidity pool and supports crypto transfers across multiple assets.

**Supported operations:** Crypto, Wallet

---

### Luno **Live**

Route regional crypto buy, sell, and wallet operations. Luno is active across multiple African markets including Nigeria and South Africa, with strong retail and business-facing APIs.

**Supported operations:** Crypto, Pay-in

---
